NagarConnect Community Issue Management System

A full stack municipal issue reporting and management platform where citizens can report public problems such as road damage, garbage overflow, and water leakage, and administrators can manage, track, and resolve them in real time.

This system enables seamless communication between citizens and municipal authorities through a centralized complaint management workflow supported by live notifications and status tracking.

Core Features
Citizen Panel

Create public service issues with priority calculation

Track issue lifecycle from submission to resolution

Receive real time system notifications when admin updates issue status

View complete issue history and audit logs

Admin Panel

Real time notification dashboard

Manage issue status: Submitted, Assigned, In Progress, Resolved, Closed

View detailed issue reports from notification panel

Delete or update reported issues

Monitor overdue issues based on SLA deadlines

Yearly analytics dashboard with charts

User and issue statistics overview

Real Time System

Socket.io based instant notifications

Admin receives alerts when a new issue is created

Citizens receive live updates when issue status changes

Notification bell with unread count and sound alert

Mark notifications as read

Detailed notification view with direct issue navigation

Admin updates issue status

Citizen receives real time notification

Project Objective

To digitalize municipal issue reporting by providing a transparent, real time complaint tracking system that improves response efficiency and citizen engagement.
